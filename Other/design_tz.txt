Проект "Simple Notes": Техническое задание для разработки дизайна Android-приложения с учетом дополнительной информации

Общее описание
Цель проекта "Simple Notes" - разработать дизайн для Android-приложения, предназначенного для создания, редактирования и удаления заметок. Приложение должно быть ориентировано на широкую аудиторию и быть комфортным для использования.

Структура приложения
Приложение будет состоять из следующих экранов:

Сплэш-скрин

Экран 1: Регистрация и авторизация
 - Форма регистрации с полями: Полное имя, Электронный адрес, Пароль
 - Форма авторизации с полями: Электронный адрес, Пароль
 - Восстановление пароля через электронную почту
 - Кнопка "Try local" или "Skip" для использования заметок оффлайн без регистрации
 * При нажатии на Try local или skip у нас появляется предупреждение о том что если юзайте оффлайн и без регистрации ваши заметки не в столь надёжном состоянии как при хрании онлайн. 

Экран 2: Список заметок
 - Отображение списка созданных заметок
 - Кнопка "Добавить заметку"
 - Кнопка "Удалить заметку" для каждой из заметок в списке
 - Кнопка "Logout" для выхода из аккаунта
 !Новое - возле кнопки "Logout" отображается имя пользователя которое мы писали при регистрации

Экран 3: Редактирование заметки
 - Поле для редактирования текста заметки
 - Кнопка "Сохранить изменения"
 - Кнопка "Удалить заметку"

Дополнительные элементы дизайна
 - Логотип приложения "Simple Notes"
 - Превью контент для страницы приложения в Google Play

Требования к дизайну

Универсальность: дизайн должен быть приятным и удобным для использования для широкого круга пользователей.
Адаптивность: дизайн должен адекватно отображаться на разных устройствах и разрешениях экрана.
Предусмотреть места для размещения баннеров.
